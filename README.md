# 🧠 Brain Tumor Classification using Deep Learning

## About
Brain Tumor Classification using Deep Learning is a web application that uses a Convolutional Neural Network (CNN) to classify brain MRI images into different tumor categories. Developed using Python, TensorFlow, Keras, and Flask, it provides fast and accurate predictions through a simple and user-friendly web interface.
